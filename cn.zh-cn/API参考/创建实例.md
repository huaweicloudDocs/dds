# 创建实例<a name="dds_api_0020"></a>

## 功能介绍<a name="section4284989"></a>

创建文档数据库实例，包括集群实例、副本集实例、以及单节点实例。

## 接口约束<a name="section1239972118494"></a>

-   该接口支持DDS社区版和增强版。
-   该接口暂不支持创建包周期实例。
